Key Terms

JavaScript (JS)

Also known as ECMAScript, the default scripting language of the web.

Operators

Include +, -, *, /, %. Symbols that tell the computer to do math.

Variables

Names to pointing to data, or functions, or basically anything.

Variable naming conventions

Don't begin with a number, or capital letter. camelCase is preferred over camel-case.

Functions

A set of operations performed on an object, may have parameters that require input when called, called using the function's name and arguments.

Methods

A kind of function that runs on a particular kind of object (e.g. strings, numbers, arrays)

Arguments

Data that is passed to a function when that function is called.

Parameters

The places in a function where arguments will go when the function is called.

Return

A value that a function sends back to where it was called from.

Chaining methods

Multiple methods acting on the same input, in order, one inside the other.

Strings

A Data Type, a set of characters inside "quotes".

Booleans

A Data Type, can only be True or False. Correspond to 1 and 0 in binary, and open or closed in electrical circuits.

Undefined

A Data Type. Javascript assigns this value to a new variable by default, before assigning any other value to it.

NaN

A value whose data type is Number. Represents a value outside of the set of numbers.

Escape

A special symbol that tells a program to treat the next character after the escape symbol differently.

Alert

A method which creates a pop up dialogue box with text or numbers and OK and Cancel buttons.

Comments

Lines of code that will not be read by the computer, but are for humans to communicate about their code. Symbols include //, /* */.

JQuery

A library of premade Javascript.

Attributes

Metadata about a file or object.
